iPad Booking Help Guide

Welcome to our iPad booking system! This help guide will walk you through the process of booking, reserving, and returning iPads using our website. Just follow these simple steps:

  1. Select Campus: Start by choosing the appropriate campus from the dropdown menu.
  2. Enter Staff Name: Type your name in the "Staff Name" field.
  3. Select iPads: Choose which iPads you've taken by confirming the numbers on the iPad itself and selecting the corresponding checkboxes.
  4. Set Return Time: Indicate how long you need the iPads by selecting an option from the "To be returned in" dropdown menu.
  5. Book iPads: Click on the "Book iPads" button to complete your booking.

To return the iPads:

  1. Select iPads: Choose the iPads you're returning by selecting the corresponding checkboxes.
  2. Return iPads: Click on the "Return iPads" button to complete the return process.

To reserve iPads for a future date and time:

  1. Follow steps 1-4 from the iPad booking process.
  2. Enter Reservation Date and Time: Select the date and time you'd like to reserve the iPads using the provided fields.
  3. Reserve iPads: Click on the "Reserve iPads" button to finalize your reservation.

That's it! You're now ready to book, reserve, and return iPads using our website.

If you notice any bugs / errors, please send a detailed email to ITService@msa.qld.edu.au (With pictures if possible).
